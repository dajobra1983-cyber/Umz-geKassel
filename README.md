# 🚚 Kraft & Konzept - Umzug & Montage Kassel

Eine moderne, responsive Single-Page-Application (SPA) für Dienstleistungen im Bereich Umzug, Möbelmontage und Entrümpelung. 

## 💡 Über das Projekt
Diese Landingpage dient als digitale Visitenkarte und Akquise-Tool. Sie wurde entwickelt, um potenziellen Kunden schnell und übersichtlich alle relevanten Dienstleistungen zu präsentieren und die Kontaktaufnahme zu vereinfachen. Ein integrierter, interaktiver JavaScript-Kalkulator filtert Anfragen vor und gibt Kunden eine erste Aufwandsschätzung.

## ⚙️ Tech Stack & Features
* **Frontend:** Reines HTML5, CSS3 und Vanilla JavaScript (Keine schweren Frameworks, Fokus auf maximale Performance und schnelle Ladezeiten).
* **Styling:** Custom CSS mit CSS-Variables, Grid/Flexbox-Layouts und Intersection Observer API für weiche Scroll-Animationen.
* **Hosting & CI/CD:** Bereitgestellt via **Azure Static Web Apps**. 
* **Deployment:** Vollautomatisierte CI/CD-Pipeline über GitHub Actions (Push-to-Deploy).

## 🚀 Lokale Ausführung
Da es sich um statische Dateien handelt, sind keine komplexen Build-Tools oder Server-Umgebungen nötig:
1. Repository klonen: `git clone https://github.com/dajobra1983-cyber/Umz-geKassel.git`
2. Die Datei `index.html` direkt im Browser öffnen oder über eine Extension wie "Live Server" in VS Code starten.

## ✉️ Kontakt & Anfragen
Konfiguriert für direkte E-Mail-Anfragen mit vorausgefüllten Betreff- und Body-Parametern (Mailto-Protokoll), um strukturierte Kundenanfragen zu gewährleisten.
